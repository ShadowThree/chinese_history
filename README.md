## E003
**新石器时代晚期**  3-5千年前
1. 以虚拟的龙作为华夏族的图腾，是因为在各个民族融合的过程中，图腾也进行了融合；
2. 皇帝(公孙轩辕)部落在黄河流域，先后发送了两次大战：逐鹿之战(蚩尤战死)，阪泉之战(炎帝臣服)。通过这两次大战，使得各民族部落逐渐融合，形成华夏族。
3. 史官仓颉造字
4. 神农尝百草（炎帝）
5. 中华的来源：皇帝部落叫华夏族，西周初年周公定洛邑为中土，而皇帝部落刚好在那里，所有称为中华民族。
